Design with Tailwind

Utility-First CSS Framework

What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework for rapidly building custom designs. Instead of writing custom CSS, you use pre-defined utility classes directly in HTML.

Core Concepts

  • Utility Classes: Predefined CSS classes for spacing, colors, typography, etc.
  • Responsive Design: Use breakpoints like sm:, md:, lg:
  • Variants: Hover, focus, dark mode, active, etc.
  • Customization: Extend default Tailwind in tailwind.config.js

Tailwind Workflow

How to use Tailwind in projects:

  1. Install Tailwind via npm or CDN
  2. Add @tailwind base;, @tailwind components;, @tailwind utilities; in CSS
  3. Use utility classes in HTML
  4. Build/compile CSS for production

Common Utility Classes

  • Spacing: p-4, m-2, mx-auto
  • Colors: bg-blue-500, text-white, hover:bg-indigo-600
  • Typography: font-bold, text-xl, uppercase
  • Flex & Grid: flex, justify-center, grid grid-cols-3
  • Effects: shadow-lg, rounded-xl, hover:scale-105

Responsive & Variants

  • Breakpoints: sm:, md:, lg:, xl:
  • Hover: hover:bg-red-500
  • Focus: focus:ring-2
  • Dark mode: dark:bg-gray-800
  • Active: active:scale-95

Best Practices

  • Use utility classes instead of custom CSS for speed
  • Keep HTML readable and structured
  • Use responsive design utilities
  • Leverage variants like hover, focus, dark mode
  • Use @apply in custom CSS if needed
