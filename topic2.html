Introduction to CSS

Cascading Style Sheets (CSS)

Introduction

CSS is a language used to describe how HTML elements should look on a web page. It controls the style of a website, such as:

  • Colors
  • Fonts
  • Layout
  • Spacing
  • Animations

What is "The Cascade"?

The cascade determines which style rules are applied when multiple rules target the same element. It depends on rule origin (author/user/agent), specificity (inline, id, class, element) and the order of rules.

Concepts

Specificity in CSS determines which rule wins when multiple rules target the same element. Each type of selector has a point value. The more specific the selector, the higher its score.

CSS Syntax

A CSS rule consists of a selector and a declaration block: 

 selector { property: value; }

Parts of a Syntax

Part Description
Selector Specifies which HTML elements to style (e.g., p, .class, #id)
Property The aspect to change (color, font-size)
Value The setting for the property (red, 16px)
Declaration A property + value pair (color: red;)
Declaration Block One or more declarations inside { ... }

Common Categories of CSS Properties

Groupings include text, box model, background, layout, flexbox/grid, animation, and visibility/effects.

  • Text: color, font-size, line-height, text-align
  • Box model: margin, padding, border, width, height
  • Background: background-color, background-image, background-position
  • Layout: display, position, float, grid, flex
  • Animation: transition, animation

Grid System

A CSS Grid System is a layout method used to create web page layouts based on rows and columns. It allows for precise control and responsive designs.

  • Grid Container
  • Grid Items
  • Grid Lines
  • Grid Tracks
  • Grid Cells
  • Grid Areas
  • Gaps (Gutters)
  • Nested Grids
  • Media Queries (Responsive Design)

Use display: grid; on a container and define columns/rows with grid-template-columns / grid-template-rows. Combine with gaps and media queries for responsive layouts.
